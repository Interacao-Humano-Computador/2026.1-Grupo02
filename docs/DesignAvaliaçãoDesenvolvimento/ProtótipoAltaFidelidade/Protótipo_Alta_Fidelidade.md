## Introdução

Este documento centraliza o acesso a todos os protótipos de alta fidelidade produzidos pelo Grupo 02 ao longo do projeto. Cada protótipo corresponde a uma tarefa identificada nos cenários de uso do sistema e foi desenvolvido como protótipo de alta fidelidade, com o objetivo de representar visualmente a estrutura das interfaces e a sequência de interações do usuário, subsidiando as etapas de design e avaliação previstas na disciplina de Interação Humano-Computador.

Os protótipos de alta estão organizados por tarefa e podem ser acessados individualmente pelos links disponíveis na Tabela 2.

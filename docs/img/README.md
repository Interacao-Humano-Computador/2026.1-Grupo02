# 🖼️ Recursos Visuais e Imagens

Esta pasta destina-se exclusivamente ao armazenamento de ativos visuais utilizados na documentação do projeto.

**Conteúdo esperado:**
* Logótipos oficiais;
* Capturas de ecrã (*screenshots*) do portal da SES-DF;
* Diagramas e fluxogramas;
* Fotos ou ilustrações para protótipos de interface.

Para manter a organização, recomenda-se a utilização de nomes descritivos e formatos leves (ex: `.png`, `.jpg` ou `.svg`).

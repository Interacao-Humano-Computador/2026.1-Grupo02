## Introdução

Este documento centraliza o acesso a todos os cenários de uso produzidos pelo Grupo 02 ao longo do projeto. Cada cenário descreve em detalhes o contexto de uso do sistema, os objetivos dos usuários e as sequências de interações esperadas, com o objetivo de subsidiar as etapas de análise de requisitos e design na disciplina de Interação Humano-Computador.
